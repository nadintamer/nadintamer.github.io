Lexis

Tools: Java, Android Studio, Parse, REST APIs | Source Code: GitHub

It can be difficult to find the motivation or time to learn a new language â€” so what if you could learn new words while reading regular internet content? Lexis is an Android language-learning app designed to be immersive and fun. It allows users to see translated words embedded within internet content for in-context learning and practice in a fun way through flashcards & word search puzzles.

I developed Lexis as my final project during Facebook University. It was definitely a challenge to develop the app within 4 weeks, but also a huge learning opportunity. I learned a lot about how to integrate different APIs within my app (I used the Wikipedia API, News API, and the Google Cloud Translate & NLP APIs!). It was also an awesome challenge to implement the computer-generated word search puzzle for vocabulary practice. Overall, I'm incredibly proud of my work on Lexis and can't wait to continue improving the app!
